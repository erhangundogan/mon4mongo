mon4mongo
==========
MongoDB database management web server boosted with node.js & express.
Project is in a development state. Libraries below are using:

+ node.js
+ express
+ jade
+ node-mongodb-native
+ bootstrap
+ jquery

You can clone github project or use npm to install.
```
npm install mon4mongo
```
